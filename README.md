# Travaux Pratiques – POO Java

Ce dépôt regroupe l’ensemble des exercices réalisés dans le cadre des **travaux pratiques (TP)** du module **Programmation Orientée Objet (POO) en Java**.

Les exercices correspondent à **trois TPs distincts**, couvrant les notions fondamentales et avancées de la POO en Java, notamment :
- Classes et objets
- Encapsulation
- Héritage
- Classes abstraites
- Interfaces
- Polymorphisme

## Contenu du dépôt

- **TP1 / TP2** : Classes et objets, gestion de scénarios simples (bibliothèque, banque, etc.)
- **TP3** : Classes abstraites et interfaces (formes géométriques, animaux et capacités)

Chaque TP est structuré en packages clairs et organisés, facilitant la lecture et l’évaluation du code.
